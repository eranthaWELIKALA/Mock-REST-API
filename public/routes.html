Mock REST API

Available Routes

/config

Add json objects using the user interface

/response

Create response with POST requests, Retrieve the selected response with GET request

{
    response: response object: Object | Array,
    name: name for the response: String,
    isSelected: whether this value should be marked as the selected response: boolean (true | false),
}

You can use UI as well to insert responses

/response/:responseId

GET request with response Id, this will return the response for the Id

/select/:responseId

GET request with response Id, this will mark the response as the selected response
